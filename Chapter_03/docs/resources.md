# 🔗 Chapter 03 Resources

Use these hand-picked resources to fully master strings.

## 📖 Official Documentation
- [Python 3: String Methods](https://docs.python.org/3/library/stdtypes.html#string-methods) - An exhaustive list of every built-in string method available in Python. Skim this to see what's possible!
- [Python 3: F-Strings](https://docs.python.org/3/reference/lexical_analysis.html#f-strings) - Deep dive into string formatting.

## 🎓 Best YouTube Explanations
- [Corey Schafer: Working with Textual Data](https://www.youtube.com/watch?v=k9TUPpGqYTo) - One of the best, most comprehensive guides on slicing and formatting strings.

## 🏋️ Interview Practice
- [LeetCode: String Tag](https://leetcode.com/tag/string/) - Once you feel confident, sort these by "Easy" and try solving a few real-world interview string manipulation problems!
