## 🎯 Overview
Strings are one of the most fundamental data structures in Python. Almost all data you scrape from the web, read from a file, or take from user input starts as a string. In this chapter, you will learn how to manipulate text, slice it into smaller pieces, format it for display, and use built-in string methods.

## 🔑 Learning Objectives
- Understand string creation (single, double, and triple quotes).
- Master Positive and Negative String Slicing (e.g., `word[1:4]`).
- Understand string immutability.
- Use built-in string functions (`.replace()`, `.find()`, `.count()`).
- Master Escape Sequence Characters (`\n`, `\t`).
- Use F-strings to dynamically format text.

## ⚠️ Prerequisites
- Variables & Data Types (Chapter 02)
- Understanding indexing (starting at 0).
