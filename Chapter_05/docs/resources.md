# 🔗 Chapter 05 Resources

If the notes in this chapter weren't enough, use these hand-picked resources to master Dictionaries and Sets.

## 📖 Official Documentation
- [Python 3: Dictionaries](https://docs.python.org/3/tutorial/datastructures.html#dictionaries) - Official syntax and methods.
- [Python 3: Sets](https://docs.python.org/3/tutorial/datastructures.html#sets) - Official set math logic.

## 🎓 Best YouTube Explanations
- [Corey Schafer: Dictionaries - Working with Key-Value Pairs](https://www.youtube.com/watch?v=daefaLgNkw0) (Excellent deep dive into methods).
- [CS50: Hash Tables](https://www.youtube.com/watch?v=nvzVHwrrub0) (If you want to understand *how* dictionaries achieve $O(1)$ time complexity internally, watch this).
