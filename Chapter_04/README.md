## 🎯 Overview
Up until now, our variables could only hold a single piece of data (like one string or one integer). **Lists and Tuples** are our first "Data Structures"—they allow us to store multiple pieces of data inside a single variable.

## 🔑 Learning Objectives
- Create Lists and access their elements using indexing.
- Understand the defining feature of Lists: **Mutability**.
- Master List Methods (`.append()`, `.sort()`, `.pop()`, `.insert()`).
- Create Tuples and access their elements.
- Understand the defining feature of Tuples: **Immutability**.
- Know when to use a List vs when to use a Tuple.
