# 🔗 Chapter 04 Resources

Use these hand-picked resources to fully master lists and tuples.

## 📖 Official Documentation
- [Python 3: Data Structures](https://docs.python.org/3/tutorial/datastructures.html) - The official Python tutorial chapter on Lists and Tuples. Extremely well written.

## 🎓 Best YouTube Explanations
- [Corey Schafer: Lists, Tuples, and Sets](https://www.youtube.com/watch?v=W8KRzm-HUcc) - A fantastic visual breakdown of when to use which data structure.

## 🛠️ Interactive Visualization
- [Python Tutor](https://pythontutor.com/) - Essential! Run a script with `.append()` and `.sort()` here. You will actually see Python creating a box in memory with pointers to other boxes. It completely demystifies how lists work behind the scenes.
