# 🚀 Chapter 04 Projects

Build this project to solidify your understanding of lists and how they store data.

## 🛒 Project 1: The Shopping Cart

**Goal:** Build a basic script that manages a user's shopping cart using List methods.
**Difficulty:** ⭐⭐☆☆☆
**Estimated Time:** 20 Mins

### Concepts Practiced
- Creating Lists
- `list.append()`
- `list.remove()`
- `list.sort()`

### Project Requirements
1. Create an empty list called `cart = []`.
2. Use `input()` to ask the user to add an item to the cart. Use `.append()` to add it to the list. Do this 3 times so the cart has 3 items.
3. Print out the cart.
4. Oh no! The user changed their mind. Ask the user for one item they want to remove. Use `.remove()` to delete it from the list.
5. Print out the cart in alphabetical order using `.sort()`.

*Note: Since you haven't learned loops yet, you will have to copy-paste the `input()` and `.append()` lines 3 times. That's okay!*
