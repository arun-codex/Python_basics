# 🧩 Chapter 01 Practice Set: Modules & Comments

This is the very first practice set. It tests your ability to write basic scripts and use modules.

## 📝 Problem Index

### Problem 1: Multi-line Printing
- **File:** [`Problem_1.py`](./Problem_1.py)
- **Difficulty:** ⭐☆☆☆☆
- **Concept Tested:** `print()` function and multi-line strings.
- **Expected Time:** 5 Mins
- **Hint:** Use three single quotes `'''` inside your print statement.
- **Common Mistake:** Using single quotes `print('line1 \n line2')` instead of the more readable `'''` syntax.
- **Revision Priority:** Low

### Problem 3: The OS Module
- **File:** [`Problem_3.py`](./Problem_3.py)
- **Difficulty:** ⭐⭐☆☆☆
- **Concept Tested:** Importing built-in modules.
- **Expected Time:** 10 Mins
- **Hint:** Use `import os` and read the documentation for `os.listdir()`.
- **Common Mistake:** Forgetting to `import os` before calling `os.listdir()`. This throws a `NameError`.
- **Revision Priority:** Medium

- **Difficulty:** ⭐☆☆☆☆
- **Concept Tested:** Single-line comments.
- **Expected Time:** 5 Mins
- **Hint:** Use the `#` symbol.
- **Revision Priority:** Low
