## 🎯 Overview
This chapter introduces the fundamental building blocks of Python. You will learn how to store information using **Variables**, the different types of information Python understands (**Data Types**), and how to manipulate that data using **Operators** and **Typecasting**.

## 🔑 Learning Objectives
- Understand what variables are and how memory allocation works abstractly.
- Memorize the strict rules for naming variables in Python.
- Differentiate between Integers, Floats, Strings, and Booleans.
- Perform Arithmetic, Assignment, Comparison, and Logical operations.
- Understand the `type()` function and type conversion (typecasting).
- Capture user input using the `input()` function.

## ⚠️ Prerequisites
- Writing a basic `.py` script (Chapter 01)
- Running a script in the terminal (Chapter 01)
