**Data Types** 

Primarily there are some types of Data Types in python

   1. Integer (int) 
   2. Floating Point Number (float) 
   3. String (str) 
   4. Boolean (bool) 
   5. None 


   1. Integer (int) : Integer are the whole numbers, positive or negative, without a decimal component. 

   Example : 1 , -1 , 100 , -100 

   2. Floating Point Number (float) : Float are the numbers with a decimal component. 

   Example : 1.5 , -1.5 , 100.5 , -100.5 

   3. String (str) : String are the sequence of characters. 

   Example : "Arun" , "Hello" , "123" , "123.5" 

   4. Boolean (bool) : Boolean are the truth values, True or False. 

   Example : True , False 

   5. None : None is a special data type that represents the absence of a value. 

   Example : None 


**How to check the type of a variable** 


We use **type()** function to check the type of a variable 

Example : type(a) 




**Rules to follow for creating a Variable in python** 

1. Variable name must start with a letter or an underscore.
2. Variable name cannot start with a number.
3. Variable name can only contain alphanumeric characters and underscores (A-z, 0-9, and _).
4. Variable names are case-sensitive.
5. Variable names cannot be Python keywords.



**Operators**


There are various types of operators in python


   1. Arithmetic Operators (+, -, *, /, %, **)
   2. Assignment Operators (=, +=, -=, *=, /=, %=, **=)
   3. Comparison Operators (==, !=, <, >, <=, >=)
   4. Logical Operators (and, or, not)
   5. Identity Operators (is, is not)
   6. Membership Operators (in, not in)
   7. Bitwise Operators (&, |, ^, ~, <<, >>)
