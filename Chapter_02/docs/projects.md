# 🚀 Chapter 02 Projects

Build these to solidify your understanding of typecasting, user input, and arithmetic operations.

## 🧮 Project 1: The Tip Calculator

**Goal:** Build a script that calculates the final bill including tip for a restaurant.
**Difficulty:** ⭐⭐☆☆☆
**Estimated Time:** 20 Mins

### Concepts Practiced
- `input()`
- Typecasting (`float()`)
- Arithmetic operations (`*`, `+`)

### Project Requirements
1. Ask the user: *"What was the total bill?"*
2. Ask the user: *"What percentage tip would you like to give? (e.g., 10, 15, 20)"*
3. Ask the user: *"How many people are splitting the bill?"*
4. Calculate the total bill (bill + tip).
5. Calculate how much each person needs to pay.
6. Print the final amount per person!

### 🌟 Bonus Feature
Look up how to format a float in Python so that the final amount always prints with exactly 2 decimal places (e.g., `$25.40` instead of `$25.4`).

## 📅 Project 2: Age in Days Calculator

**Goal:** Convert a user's age into days, weeks, and months.
**Difficulty:** ⭐☆☆☆☆
**Estimated Time:** 15 Mins

### Concepts Practiced
- Basic Arithmetic (`*`)
- F-Strings (or string concatenation)

### Project Requirements
1. Ask the user for their current age in years.
2. Calculate how many days, weeks, and months they have been alive.
   - *Assume 365 days in a year, 52 weeks in a year, and 12 months in a year.*
3. Print out a message like: *"You have been alive for X days, Y weeks, and Z months!"*
