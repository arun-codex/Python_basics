# 🧠 Active Recall Flashcards

Use these for your 5-Minute Revision. Cover the answer, read the question, and force yourself to recall the answer before checking.

---

<details>
<summary><b>Question:</b> Are variable names in Python case-sensitive?</summary>
<br>
<b>Answer:</b> Yes. `Age`, `age`, and `AGE` are three different variables.
</details>

<details>
<summary><b>Question:</b> Can a variable name start with a number? (e.g., `1_name`)</summary>
<br>
<b>Answer:</b> No. Variables must start with a letter or an underscore.
</details>

<details>
<summary><b>Question:</b> What does the `type()` function do?</summary>
<br>
<b>Answer:</b> It returns the data type of the variable passed into it (e.g., `<class 'int'>`).
</details>

<details>
<summary><b>Question:</b> What data type does the `input()` function ALWAYS return?</summary>
<br>
<b>Answer:</b> A String (`str`), even if the user types a number.
</details>

<details>
<summary><b>Question:</b> What is the difference between `=` and `==`?</summary>
<br>
<b>Answer:</b> `=` is the Assignment Operator (puts a value into a variable). `==` is the Comparison Operator (checks if two values are equal and returns a Boolean).
</details>

<details>
<summary><b>Question:</b> What is the output of `5 / 2` versus `5 // 2`?</summary>
<br>
<b>Answer:</b> `5 / 2` is `2.5` (Float). `5 // 2` is `2` (Integer, floor division).
</details>
