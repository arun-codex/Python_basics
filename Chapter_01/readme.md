## 🎯 Overview
This is the foundational chapter where you write your first lines of Python. You will learn how to interact with Python, how to use built-in and external modules using `pip`, and how to document your code with comments.

## 🔑 Learning Objectives
- Write and execute a basic Python script.
- Understand the difference between the REPL (Read-Eval-Print Loop) and script files.
- Learn to install external packages using `pip`.
- Understand built-in modules vs external modules.
- Write single-line and multi-line comments.

## ⚠️ Prerequisites
- Python installed on your computer.
- A basic understanding of opening a terminal/command prompt.
