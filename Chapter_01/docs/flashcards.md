# 🧠 Active Recall Flashcards

Use these for your 5-Minute Revision. Cover the answer, read the question, and force yourself to recall the answer before checking.

---

<details>
<summary><b>Question:</b> What does REPL stand for?</summary>
<br>
<b>Answer:</b> Read-Eval-Print Loop.
</details>

<details>
<summary><b>Question:</b> How do you exit the Python REPL?</summary>
<br>
<b>Answer:</b> Type `exit()` and hit enter.
</details>

<details>
<summary><b>Question:</b> What is the difference between an external and built-in module?</summary>
<br>
<b>Answer:</b> Built-in modules come with Python (like `os` or `math`). External modules must be downloaded from the internet using `pip`.
</details>

<details>
<summary><b>Question:</b> Is `pip install` a Python command?</summary>
<br>
<b>Answer:</b> No! It is a terminal/command-line command. You cannot run it inside a `.py` script or inside the Python REPL.
</details>

<details>
<summary><b>Question:</b> Does the Python Interpreter read multi-line comments starting with `'''`?</summary>
<br>
<b>Answer:</b> Yes. They are technically multi-line strings. However, because they aren't assigned to a variable, Python evaluates them and instantly throws them away.
</details>
