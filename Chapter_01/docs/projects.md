# 🚀 Chapter 01 Projects

Because this is the very first chapter, you don't have enough logic (if/else, loops) to build complex projects yet. But you can still build something cool by relying entirely on external modules!

## 🗣️ Project 1: The Talking Python

**Goal:** Build a script that speaks whatever text you write.
**Difficulty:** ⭐⭐☆☆☆
**Estimated Time:** 15 Mins

### Concepts Practiced
- Using `pip` to install an external library.
- Importing a module.
- Writing good comments.

### Project Requirements
1. Open your terminal and run `pip install pyttsx3`.
2. Create a new file called `talking_bot.py`.
3. Read the documentation online to figure out how to import `pyttsx3`, initialize it, and use the `.say()` and `.runAndWait()` methods.

### 🌟 Bonus Feature
Add comments explaining exactly what each line of the `pyttsx3` initialization does so Future Arun doesn't forget!
