# ⚡ Chapter 01 Cheat Sheet: Modules, Comments & pip

Use this page for rapid 5-minute and 15-minute revisions right before an exam.

## 📌 1. Python Commands

| Action | Terminal Command | Notes |
| :--- | :--- | :--- |
| **Start REPL** | `python` | Opens the interactive shell. |
| **Exit REPL** | `exit()` | Closes the interactive shell. |
| **Run a Script** | `python filename.py` | Executes a saved Python file. |
| **Install Module** | `pip install modulename` | Downloads external libraries. |

## 📌 2. Writing Code & Comments

| Type | Syntax | Example |
| :--- | :--- | :--- |
| **Print to screen** | `print()` | `print("Hello World")` |
| **Single-line Comment**| `#` | `# This is a comment` |
| **Multi-line Comment** | `'''` or `"""` | `''' This is a paragraph '''` |
| **Importing Module** | `import` | `import os` |

## 🚨 Things I Always Forget
1. `pip install` is typed into the **Terminal (Command Prompt / PowerShell)**, NOT into the Python code editor.
2. The print statement in Python 3 requires parentheses: `print("Hello")`.
3. To play an mp3 file in Python, you can use the `playsound` external module (which you must pip install first).
