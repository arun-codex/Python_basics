# 🧩 Chapter 04 Practice Set: Lists & Tuples

This practice set heavily tests your ability to manipulate data structures using `.sort()`, `.append()`, and `.count()`.

## 📝 Problem Index

### Problem 1: Store User Input in a List
- **File:** [`Problem_01.py`](./Problem_01.py)
- **Difficulty:** ⭐⭐☆☆☆
- **Concept Tested:** `input()`, `list.append()`.
- **Expected Time:** 10 Mins
- **Hint:** Create an empty list first `[]`, then ask for input, then `.append()` to the list.

### Problem 2: Sort Marks
- **File:** [`Problem_02.py`](./Problem_02.py)
- **Difficulty:** ⭐⭐☆☆☆
- **Concept Tested:** `.sort()`.
- **Expected Time:** 5 Mins
- **Common Mistake:** `sorted_list = marks.sort()` will give you `None`. Remember that `.sort()` modifies the list in place!
