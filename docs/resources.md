# 📚 Python Learning Resources

A curated list of the absolute best resources to supplement my learning. If I get stuck on a topic, I will check here first.

## 📖 Official Documentation
- [Python 3 Official Docs](https://docs.python.org/3/) - The source of truth.
- [PEP 8 Style Guide](https://peps.python.org/pep-0008/) - How to write clean, professional Python.

## 🎓 Video Tutorials
- [Corey Schafer's Python Series](https://www.youtube.com/playlist?list=PL-osiE80TeTt2d9bfVyTiXJA-UTHn6WwU) - Deep dives into Python internals.
- [Programming with Mosh](https://www.youtube.com/watch?v=_uQrJ0TkZlc) - Great for quick syntactical overviews.

## 🏋️ Practice & Interview Prep
- [LeetCode](https://leetcode.com/) - Essential for Data Structures & Algorithms.
- [HackerRank](https://www.hackerrank.com/domains/python) - Good for mastering basic Python syntax.
- [NeetCode](https://neetcode.io/) - Roadmap for LeetCode blind 75.

## 🛠️ Interactive Environments
- [Python Tutor](https://pythontutor.com/) - Visually execute Python code step-by-step. Extremely helpful for understanding loops and memory references.
